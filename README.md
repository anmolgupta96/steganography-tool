# Steganography Tool 🔐

A Python cybersecurity tool to hide and reveal 
secret messages inside images using LSB 
Steganography with password protection.

## Features
- Hide secret messages in images
- Password protection
- Reveal messages with correct password
- Professional dark UI

## How to Use
Download SteganographyTool.exe from dist folder
and double click to run!
